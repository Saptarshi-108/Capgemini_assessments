# Problem: Library.java – Simple Library Management System

## Problem Statement (Simple LLD / OOP)

Design a simple in-memory library system that:

Manages Books with bookId, title, and author.

Tracks which books are available vs issued to Members.

A Member has a memberId and name.

Requirements:

Add books to library inventory.

Issue a book to a member (if available).

Return a book (makes it available again).

Check book availability and member borrowing history.

OOP Focus: Encapsulation, composition (Library owns Books and tracks Members).
